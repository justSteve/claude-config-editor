## Adding New Documents

When adding new agent-generated markdown files, follow this naming convention:

```
{category}-{descriptive-name}.md
```

**Examples:**
- `design-search-feature-architecture.md`
- `documentation-cli-commands.md`
- `planning-phase-3-roadmap.md`
- `session-summary-reporting-implementation.md`
- `marketing-hackernews-launch.md`

This keeps documentation organized, searchable, and easy to navigate.
